What is CSS?

CSS is a stylesheet language used to describe the presentation of a document written in HTML. It controls how the content looks, not what it says.

Types of CSS

Inline CSS

Style applied directly to an HTML tag

Internal CSS

CSS inside tags in the HTML files Head

External CSS

Linked using a .css file

Cascading Order (Priority)

If styles conflict, CSS follows this order of priority:

Inline CSS
Internal CSS
External CSS
Browser default styles

selectors:

selectors are patterns used to select the elements you want to style

Type Selector (Element Selector):Selects all elements of a given tag name.

ID Selector:Selects an element based on its id attribute.Prefix with #

Class Selector:Selects elements based on the class attribute.Prefix with.

Universal Selector:Selects all elements.

Group Selector:Selects multiple types of elements and applies the same style.
